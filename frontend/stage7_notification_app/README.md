# Stage 7 Notification App

This is a responsive React frontend implementation for the notification dashboard.

## Features

- Displays all notifications
- Shows top 10 unread priority notifications
- Filters notifications by type
- Distinguishes new and viewed notifications
- Responsive layout for mobile and desktop

## Run locally

```bash
cd frontend/stage7_notification_app
npm install
npm run dev
```

Open the local Vite URL shown in the terminal.
